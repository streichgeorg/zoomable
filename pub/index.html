Zoomable.js

It's a javascript library that lets developers create zoomables user interfaces while writing normal HTML. An example how the library could be used can be found here.

Usage

The main way a developer can interact with the library is through the special zoomable css classes, which specify the layout mode that should be used for that element. Children of elements tagged with such a class are scaled to fit into the parent element in the setup step. The library itself provides two methods, the first one is setup which when passed an element transforms all the tagged elements. The other method that is provided is focus which given the id of an element focusses it.

Layout Modes

Currently there are two layout modes implemented. The class zoomable indicates the normal layout mode which just scales down the element so that it fits in both the vertical and horizontal dimension. The second mode that can be used is the text layout mode indicated by the class zoomable-text. This assumes that the content inside the element can reflow in an area preserving way when the aspect ratio changes.
